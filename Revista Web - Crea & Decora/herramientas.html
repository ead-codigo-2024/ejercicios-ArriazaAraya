Herramientas y materiales para dibujar tus proyectos de lettering

Una de las muchas ventajas del lettering es que puedes utilizar cualquiera de las herramientas cotidianas que tienes a mano para hacer un dibujo

Cada diseño se proyecta para un soporte concreto y, por tanto, las herramientas y materiales que usaremos dependerán de este soporte.

Por ejemplo, cuando pienses en una postal, necesitarás colores para papel. En cambio, cuando tengas en mente un diseño para el escaparate de tu tienda, deberás usar pintura para vidrio. Es decir, el abanico de materiales que podemos usar para dibujar el lettering es inmenso. Aquí tienes un listado de los básicos indispensables que necesitarás para dibujar.

Lápiz y portaminas

Son las herramientas básicas con las que empezar cualquier boceto. Los lápices y minas vienen marcados con la letra H (duro) o B (blando). Lo ideal para bocetar es un 2H y un HB (medio).

Goma de borrar

Cuanto menos lo usemos mejor, pero sin duda deberás tenerla a mano. Es recomendable tener una goma estándar y otra en forma de lápiz para pequeños detalles.

Regla pequeña y compás

Solo están pensados para crear líneas base y delimitar espacios curvos (nunca para dibujar las letras del lettering).

Lápices de colores

Tanto para bocetar como para el acabado final. Además, los hay normales o tipo acuarela (al mezclarlos con agua, se consiguen efectos muy bonitos).

Rotulador negro para papel

Es indispensable uno de punta redonda (el habitual), para pintar el interior de las letras en cada boceto ya que así se visualiza mejor el conjunto del diseño.

Rotuladores de colores para papel

Dependiendo del diseño, se pueden usar: rotuladores de punta de pincel (funcionan muy bien a la hora de pintar), rotuladores de punta biselada (para cubrir grandes superficies), rotuladores de punta flexible (para detalles), rotuladores de punta fina (para detalles muy pequeños) y rotuladores normales de colores.

Pinceles y brochas para varios productos

Brochas y pinceles planos, redondos, finos y gruesos para usarlos con acuarela, tinta, aguada, acrílicos, etc.

Rotuladores multimaterial

Existen rotuladores para diferentes soportes: madera, vidrio, cerámica, metal, etc.

Diferentes tipos de papel

Cada papel tiene sus propias características y reacciona de forma diferente ante los distintos procesos. Por este motivo, es ideal tener diferentes variedades a mano para usarlas en función del material a aplicar.

Para bocetar y practicar, sirve cualquier papel de 80-100 g liso-suave o con pequeños puntitos.

Para trabajos finales, utiliza un papel entre 150-300 g. Para técnicas líquidas, es ideal un papel de acuarela (con o sin textura) y para técnicas secas (como rotulador y lápices), puedes usar un papel compacto de algodón.

Papel de calcar

Puedes usar papel vegetal, sulfurizado o de calco (negro).

Otros soportes

El límite lo pones tú: puedes crear sobre una pieza de madera, una ventana, una pared…
